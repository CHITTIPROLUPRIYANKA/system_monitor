# Cloud-Based System Monitoring & Alert Platform

## 📌 Overview

This project is a cloud-based system monitoring platform that tracks real-time system performance metrics such as CPU and memory usage. It provides a web-based dashboard for remote monitoring.

## 🚀 Features

* Real-time CPU and memory monitoring
* Web-based dashboard
* Cloud deployment
* Scalable for multi-system monitoring

## 🛠️ Tech Stack

* Backend: Python
* Frontend: HTML, CSS, JavaScript
* Deployment: Railway

## 📊 Future Enhancements

* Multi-system monitoring
* Alerts and notifications
* Data visualization with graphs
* User authentication
